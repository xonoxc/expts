# expts

A collection of experimental projects.

## Projects

### [experience.chat](./experience.chat/)
A Next.js ai chat application that was supposed to tailor chat responses from ai
into experiences.

### [jobpipelining](./jobpipelining/)
A job processing system with real-time SSE updates. Bun + Hono + Drizzle + SQLite with React and Svelte UIs.

### [linko-starter](./linko-starter/)
A small, realistic Go project for practicing observability (logs, metrics, traces).
